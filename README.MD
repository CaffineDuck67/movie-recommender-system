# 🎬 Movie Recommender System

A simple machine learning-based movie recommendation system built using **Streamlit** and **Cosine Similarity**. It suggests movies similar to the one selected by the user and displays their posters using the **TMDB API**.

---

## 🚀 Features
- Movie recommendations based on similarity
- Clean Streamlit UI
- Fetch movie posters using TMDB API
- Fast and simple interface

---

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit
- Requests

---

## ▶️ How to Run

```bash
pip install streamlit pandas scikit-learn requests
streamlit run app.py
